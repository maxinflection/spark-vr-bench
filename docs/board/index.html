Harness

Decoding

Pool A · agentic security

Loop
300-turn canonical agent loop, openai-chat tool-call protocol, deterministic scoring.
CyberGym
per-task PoC execution; pass if PoC reproduces a deterministic vulnerability trigger.
SEC-bench
differential sanitizer on patched build; stock = pre-patch, patched = post.
CVE-Bench
sh /evaluator/done.sh return code; n locked when driver is unblocked.
ExploitBench-14
16-flag capability bitmap, T1–T5 reach. Most rows pending — full sweep blocked on cost.
